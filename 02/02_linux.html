Linux

Introduction

iRIC-MI can be built on Linux. When you want to use iRIC-MI on public cloud like AWS, Azure, Google Cloud Platform, or HPC whose OS is Linux, please refer to this section.

As an example, the procedure to install iRIC-MI on Ubuntu 22.04 is introduced.

You can download Ubuntu from the URL below:

https://ubuntu.com/download

In this section, pymt is also installed, so that BMI models can be executed connected with iRIC models.

Please refer to the URL below for the information of pymt and BMI.

  • https://pymt.readthedocs.io/en/latest/

  • https://bmi.readthedocs.io/en/stable/

Install libraries

Install the libraries needed to build iriclib and iRIC-MI library. Input the commands in Listing 1.

Listing 1 Commands for installing libraries
1sudo apt-get update
2sudo apt-get install -y libvtk9-dev
3sudo apt-get install -y libpoco-dev
4sudo apt-get install -y libmpich-dev mpich
5sudo apt-get install -y swig
6sudo apt-get install -y cmake
7sudo apt-get install -y qtbase5-dev

Install pymt using mambaforge

Install pymt using mambaforge.

mambaforge is a package for installing mamba quickly. mamba is a fast cross-platform package manager, that can be used instead of conda.

Please refer to the following URL for detail about mamba.

https://mamba.readthedocs.io/en/latest/index.html

Input the commands in Listing 2

Listing 2 Commands for installing mambaforge
1curl -L -O "https://github.com/conda-forge/miniforge/releases/latest/download/Mambaforge-$(uname)-$(uname -m).sh"
2bash Mambaforge-$(uname)-$(uname -m).sh

When commands in Listing 2 is executed, prompts to select prefix is shown. Please input /opt/mambaforge.

Input the commands in Listing 3

Listing 3 Commands for installing pymt
1/opt/mambaforge/bin/mamba install -c conda-forge pymt

Add the line in Listing 4 to ~/.bashrc.

Listing 4 The line to be added to .bashrc
1export PATH=/opt/mambaforge/bin:$PATH

Logout then login again, to apply the change to PATH.

Install iriclib

Install iriclib with the steps below:

  1. Clone iriclib repository with git

  2. Edit CMakeLists.txt

  3. Build and install iriclib

Clone iriclib repository with git

1git clone https://github.com/i-RIC/iriclib_v4.git
2git checkout iricmi

Edit CMakeLists.txt

Change the find_package section at line 43, to the following.

1find_package(Python 3.10 EXACT REQUIRED COMPONENTS Interpreter Development)

Change the target_include_directories section at line 277 to 283, to the following.

1# iriclib include directories
2target_include_directories(iriclib
3  PUBLIC
4    "$<BUILD_INTERFACE:${PROJECT_SOURCE_DIR}>"
5    "$<BUILD_INTERFACE:${PROJECT_SOURCE_DIR}/private>"
6    "$<INSTALL_INTERFACE:${CMAKE_INSTALL_INCLUDEDIR}>"
7    ${HDF5_INCLUDE_DIRS}
8)

Change the target_link_libraries section at line 285 to 290, to the following.

1# iriclib link libraries
2target_link_libraries(iriclib
3  PRIVATE
4    ${HDF5_LIBRARIES}
5    Poco::Foundation
6)

Note

It will be easier to install, by improving CMakeList.txt. When improved, we will not need to edit CMakeList.txt.

Build and install iriclib

Install iriclib with the commands in Listing 5

Listing 5 The commands to install iriclib
 1cd iriclib_v4
 2cmake \
 3  -D PocoFoundation_DIR:STRING=/usr/lib/x86_64-linux-gnu/cmake/Poco/ \
 4  -D HDF5_INCLUDE_DIRS:STRING=/usr/include/hdf5/serial \
 5  -D HDF5_LIBRARIES:STRING=/usr/lib/x86_64-linux-gnu/hdf5/serial/lib/libhdf5.so \
 6.
 7make
 8sudo make install
 9sudo ldconfig
10sudo cp /usr/local/bin/Lib/site-packages/* /opt/mambaforge/lib/python3.10/site-packages

Build and install iRIC-MI

Install iRIC-MI with the steps below:

  1. Clone iricmilib repository with git

  2. Build and install iRIC-MI library

  3. Build and install iricmi_server

  4. Install iricmi_run

  5. Install sample solvers

  6. Add environmental variables

Clone iricmilib repository with git

1git clone https://github.com/kskinoue0612/iricmilib.git

Build and install iRIC-MI library

 1cd iricmilib/iricmilib/src
 2cmake \
 3  -D PocoFoundation_DIR:STRING=/usr/lib/x86_64-linux-gnu/cmake/Poco/ \
 4  -D PocoXML_DIR:STRING=/usr/lib/x86_64-linux-gnu/cmake/Poco/ \
 5  -D HDF5_INCLUDE_DIRS:STRING=/usr/include/hdf5/serial \
 6  -D HDF5_LIBRARIES:STRING=/usr/lib/x86_64-linux-gnu/hdf5/serial/libhdf5.so \
 7.
 8make
 9sudo make install
10sudo ldconfig
11sudo cp iricmi.f90 /usr/local/src
12cd ../bindings/python
13swig -c++ -python -cppext cpp iricmi.i
14g++ -fPIC -shared -o _iricmi.so iricmi_wrap.cpp -I/opt/mambaforge/include/python3.10 -I../../src -liricmi /opt/mambaforge/lib/libpython3.10.so
15sudo cp _iricmi.so /opt/mambaforge/lib/python3.10/site-packages
16sudo cp iricmi_custom.py /opt/mambaforge/lib/python3.10/site-packages/iricmi.py

Build and install iricmi_server

1cd ../../iricmi_server
2g++ -c main.cpp
3g++ -c server.cpp
4g++ -o iricmi_server *.o -liricmi
5
6sudo mkdir -p /opt/iric/bin
7sudo copy iricmi_server /opt/iric/bin

Install iricmi_run

1cd ../tools
2mv iricmi_run.ubuntu.py iricmi_run
3chmod a+x iricmi_run
4sudo cp iricmi_run /opt/iric/bin

Install sample solvers

1cd ../testmodels
2sudo mkdir /opt/iric/solvers
3sudo cp -R global_receive_python /opt/iric/solvers
4sudo cp -R global_send_python /opt/iric/solvers
5sudo cp -R grid_att_receive_python /opt/iric/solvers
6sudo cp -R grid_att_send_python /opt/iric/solvers
7sudo cp -R grid2d_edge_python /opt/iric/solvers
8sudo cp -R grid2d3d_python /opt/iric/solvers

Note

In the example above, only the sample models developed with python are installed.

You can also install sample models developed with C++ and FORTRAN. Before installing, you need to build the source code.

You can build C++ models with g++, with -liricmi. The command below builds global_receive_c++.

1g++ -o global_receive_c++ main.cpp -liricmi

You can build FORTRAN models with gfortran, with -iricmi, after compiling iricmi.f90. The command below builds global_receive_fortran.

1gfortran -c /usr/local/src/iricmi.f90
2gfortran -o global_receive_fortran main.f90 iricmi.o -liricmi

Add environmental variables

Add the line in Listing 6 to ~/.bashrc.

Listing 6 The line to be added to .bashrc
1export PATH=/opt/iric/bin:opt/mambaforge/bin:$PATH
2export IRICMI_ROOT_PATH=/opt/iric
3export IRICMI_LOG_LEVEL=WARNING

Test run

Run model to check that iRIC-MI and sample models are installed successfully.

cd to testdata/grid2d_edge_python, and input the command below:

1iricmi_run
