Running BMI model exchanging data with iRIC models

Run BMI model exchaning data with iRIC models.

Listing 48 shows the code to run CEM as a model that exchange data with iRIC models.

Listing 48 Code to run CEM with iRIC models
1import iricmi_pymt_util
2import pymt.models
3
4cem = pymt.models.Cem()
5args = cem.setup(number_of_rows=100, number_of_cols=200, grid_spacing=200.0)
6cem.initialize(*args)
7iricmi_pymt_util.buildCgns(cem, '/home/ubuntu/iricmilib/iricmi_pymt_util/template.cgn', '/home/ubuntu/iricmilib/iricmi_pymt_util/cem.cgn')

Source code of buildcgns.py that generates CGNS file is shown in Listing 49.

Note

For CEM, there was a limitation below:

  • Calling get_grid_x() and get_grid_y() caused exception. It seems that CEM does not support reading grid coordinates. through BMI. grid_ndim() and grid_shape() was functional, so a rectangular dummy grid is output to CGNS file.
