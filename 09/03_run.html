Generates CGNS file

Generates CGNS file, that iRIC-MI library will load to get information of grid and calculation condition.

Listing 50 shows the code to generates CGNS file for CEM.

Listing 50 Code to generate CGNS file for CEM
1import iricmi_pymt_util
2import pymt.models
3
4cem = pymt.models.Cem()
5args = cem.setup(number_of_rows=100, number_of_cols=200, grid_spacing=200.0)
6cem.initialize(*args)
7
8iricmi_pymt_util.runModel(cem)

Source code of run.py is shown in Listing 51.

Note

For CEM, it was not possible to get output data defined at grid nodes.
