iRICMI_Read_Grid3d_Str_Size

Reads the size of a structured grid (the number of nodes in I direction and J direction) from the CGNS file.

Format (FORTRAN)

call iRICMI_Read_Grid3d_Str_Size(isize, jsize, ksize, ier)

Format (C/C++)

ier = iRICMI_Read_Grid3d_Str_Size(isize, jsize, ksize)

Format (Python)

isize, jsize, ksize = iRICMI_Read_Grid3d_Str_Size()
