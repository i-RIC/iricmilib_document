iRICMI_Read_Grid_Functional_Integer_Cell_WithGridId

Reads the values of functional integer type grid attribute defined at grid Cells from the CGNS file.

Format (FORTRAN)

call iRICMI_Read_Grid_Functional_Integer_Cell_WithGridId(gid, name, dimid, v_arr, ier)

Format (C/C++)

ier = iRICMI_Read_Grid_Functional_Integer_Cell_WithGridId(gid, name, dimid, v_arr)

Format (Python)

v_arr = iRICMI_Read_Grid_Functional_Integer_Cell_WithGridId(gid, name, dimid)
