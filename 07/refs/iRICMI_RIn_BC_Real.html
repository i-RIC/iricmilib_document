iRICMI_RIn_BC_Real

Register the memory address to receive the value of real number variable (in double precision) of a boundary condition.

Format (FORTRAN)

call iRICMI_RIn_BC_Real(type, num, name, val, ier)

Format (C/C++)

ier = iRICMI_RIn_BC_Real(type, num, name, val)

Format (Python)

val = iRICMI_RIn_BC_Real(type, num, name)
